# Test task before applying for technical interview

## Environment

- Ubuntu 18.04 LTS
- g++-8 version 8.4.0
- CMake version 3.10.2
- Google Test version 1.7.0

## Directories

- /src - source code
- /tst - tests

## Build and run

    mkdir build
    cd build
    cmake ..
    make all
    ./tst/ntest_tst
    ./src/ntest_run
